# Chanel & Armani – V16 Remastered Complete

Vollständig neu aufgebaute, eigenständige HTML5/PWA-Version.

Enthalten:
- drei spielbare Abschnitte einschließlich Bossarena
- Chanel und Armani mit Figurenwechsel
- Mäuse, Katzen, Löwen und Hasen aus Rohren
- begehbare Rohre
- Schwierigkeitsgrade Leicht, Mittel und Schwer
- Löwenkönig in Leicht/Mittel
- Bärenkönig als Hauptboss in Schwer
- Bossangriffe: Prankenhieb, Sprung und Schockwelle
- Touch-, Tastatur- und Offline-Unterstützung

Start: `index.html` öffnen. Für die Offline-Installation über einen lokalen Webserver oder GitHub Pages bereitstellen.
