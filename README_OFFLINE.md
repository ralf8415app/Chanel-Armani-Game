# Chanel & Armani – V13 OFFLINE

Diese Version läuft als installierbare PWA auf:

- iPhone
- iPad
- Android
- Windows 10/11

## GitHub Pages

Alle Dateien aus diesem ZIP in das Repository hochladen. Danach öffnen:

`https://ralf8415app.github.io/Armani/?v=13`

Im Menü muss **V13 OFFLINE** stehen.

## iPhone und iPad

1. Seite in Safari öffnen.
2. Teilen-Symbol antippen.
3. **Zum Home-Bildschirm** wählen.
4. **Hinzufügen**.
5. Das Spiel einmal vollständig mit Internet starten.
6. Danach lässt es sich über das App-Symbol offline öffnen.

Hinweis: iOS zeigt keinen normalen Installieren-Button wie Android. Die Installation erfolgt über Safari.

## Android

1. Seite in Chrome öffnen.
2. Menü mit den drei Punkten öffnen.
3. **App installieren** oder **Zum Startbildschirm hinzufügen** wählen.
4. Danach funktioniert das Spiel offline.

## Windows 10/11

1. Seite in Microsoft Edge oder Chrome öffnen.
2. In der Adressleiste das App-/Installieren-Symbol anklicken.
3. **Installieren** wählen.
4. Das Spiel erscheint im Startmenü und kann offline gestartet werden.

## Technische Hinweise

- Alle Spiel-Dateien werden lokal im Browser-Cache gespeichert.
- Highscores und lokale Spielstände bleiben im lokalen Speicher des Geräts.
- Nach einer neuen Version die Seite einmal mit Internet öffnen, damit der Cache aktualisiert wird.
