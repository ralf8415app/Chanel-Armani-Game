# Chanel & Armani – Weltreise V18

## Neu
- Vier vollständig neu gestaltete Länderwelten: Italien, Frankreich, Spanien und Malediven.
- Chanel ist ein brauner Teddybär mit Schleife.
- Armani ist ein Panda mit roter Fliege.
- Beide Figuren spielen gemeinsam; mit Q oder dem Wechselknopf wird die aktive Figur getauscht.
- Landesbezogene Kulissen, Farben, Wahrzeichen und Musikmotive.
- Offline/PWA-fähig und für GitHub Pages vorbereitet.
