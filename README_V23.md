# Chanel & Armani V23 – World Tour & Memory

Built directly from V19/V22 mechanics.

Countries: Italy, France, Spain, Iran, Turkey, Maldives.
Each country has landmark information and a six-pair memory game.
The Italy memory uses a custom SVG illustration of the actual Leaning Tower of Pisa.
Turkey includes animated hot-air balloons. Main interface and educational text are in English with smaller typography.
