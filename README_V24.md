# Chanel & Armani V24 – Learning & Piggyback

- All learning landmarks remain visible in each country level.
- Discovered landmarks receive a check mark and keep their label in the scene.
- Every open or matched Memory card shows the landmark name below the picture.
- Piggyback mode: press E, the CARRY button, or the bear button when both characters are close.
- Press again to put the rider down. Carrying gives a small jump boost.
- Q or SWITCH changes the active character when nobody is being carried.
