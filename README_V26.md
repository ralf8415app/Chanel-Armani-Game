# Chanel & Armani – World Tour Deluxe V26

## Neu in V26
- professioneller Ladebildschirm und erweitertes Startmenü
- Fortsetzen-Funktion und automatische lokale Speicherung
- Sammelalbum für entdeckte Sehenswürdigkeiten
- sechs Erfolge
- Datenschutz- und Impressumsbereich
- vier Sprachen: Deutsch, Englisch, Französisch und Persisch
- eigenständig algorithmisch erzeugte Musik für jedes Land
- passende, ebenfalls algorithmisch erzeugte Umgebungsgeräusche
- weicher Neustart der Musik beim Länderwechsel
- Lautstärke und Musikschalter über die vorhandenen Einstellungen

## Musik und Rechte
Die V26-Musik wird im Browser zur Laufzeit aus Oszillatoren und synthetisch erzeugtem Rauschen erstellt. Es werden keine aufgenommenen Songs, Samples oder externen Musikdateien verwendet. Vor einer kommerziellen Veröffentlichung sollten trotzdem Impressum, Datenschutz und sämtliche verwendeten Grafiken abschließend rechtlich geprüft werden.

## Installation
Das Spiel über einen HTTPS-Webserver oder GitHub Pages bereitstellen. Danach kann es unter iPhone/iPad über Safari „Zum Home-Bildschirm“, unter Android über Chrome „App installieren“ und unter Windows über Edge/Chrome als App installiert werden.
