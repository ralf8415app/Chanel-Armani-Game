# V27 Phase 2 – Hotfix 3

Dieser Hotfix ersetzt den Memory-Weiter-Button technisch vollständig, damit keine alten oder mehrfach registrierten Ereignis-Handler mehr ausgeführt werden.

Behoben:
- Italien-Memory wechselt direkt und zuverlässig nach Frankreich.
- Kein unbeabsichtigter Rücksprung auf die Weltkarte.
- Alle Länderübergänge 1→2 bis 6→7 verwenden jetzt den zentralen, vollständigen Länderstart.
- Deutschland und Schweiz öffnen nach dem Memory weiterhin ihr Quiz.
- Stempel, Fortschritt und Freischaltung werden vor dem Übergang gespeichert.
- Doppelklick und Touch-Doppelereignisse werden blockiert.
- Offline-Cache auf Hotfix 3 aktualisiert.
